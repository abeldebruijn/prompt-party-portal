## Games

This project currently includes 3 playable **Game Modes** for a shared **Lobby**:

- **Pick text that suits a situation**: A fast, funny **Game Mode** where each **Round** throws Players into a prompt and pushes them to land the sharpest text answer. Most Players submit an anonymous **Text Submission**, one **Judge** sits that Round out, and the best mix of correctness and creativity climbs the **Leaderboard**.
- **Pick image that suits a situation**: This **Game Mode** turns every **Round** into a visual showdown. Players write an **Image Prompt**, generate and regenerate previews until the idea clicks, then submit their best shot while one **Judge** scores the anonymous results and pushes the strongest entries up the **Leaderboard**.
- **Feed It Forward**: This is the wild chain-based **Game Mode** where Players seed impossible scenes, pass images around the circle, and keep guessing the hidden prompt from the latest **Source Image** they see. As the **Session** rolls on, prompts drift, scores stack on the **Leaderboard**, and the full chaotic reveal lands in the final **Chain Gallery**.
